# Static Web Hosting: Hola Mundo 🌐

Este repositorio contiene un proyecto base desarrollado para explorar, validar y configurar entornos de **Alojamiento Web (Hosting)** en la nube. El propósito central de este laboratorio es implementar el ciclo de publicación completo de un artefacto estático sencillo, asegurando que los servidores de producción sirvan y expongan el contenido correctamente hacia el internet público.

Sirve como una plantilla limpia para testear la conectividad, configuraciones de DNS, o herramientas de publicación rápida de archivos.

---

## ⚙️ Conceptos de Infraestructura Aplicados

*   **Servicio de Contenido Estático:** Comprensión de cómo un servidor web (o CDN) procesa y distribuye archivos tradicionales sin necesidad de un motor de ejecución en el backend.
*   **Validación de Entornos de Producción:** Uso de un artefacto mínimo viable para comprobar que las rutas de despliegue y los permisos de lectura sean correctos.
*   **Accesibilidad y Enrutamiento:** Configuración del acceso global al recurso mediante la URL pública proveída por el servicio de alojamiento.

---

## 📁 Estructura del Repositorio

*   **`index.html`**: El documento principal de la aplicación. Contiene la estructura básica en HTML encargada de renderizar el mensaje de "Hola Mundo" para confirmar visualmente el éxito del despliegue en el navegador.

---

## 🧰 Tecnologías Utilizadas

*   **Lenguaje de Marcado:** HTML5.
*   **Entorno:** Plataformas de Hosting Estático (ej. GitHub Pages, Vercel, Netlify o Surge).
